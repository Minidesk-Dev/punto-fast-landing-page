# Punto Fast Landing Page

Landing page estática para Punto Fast, incluido el build de Netlify y archivos de recursos.

## Estructura

- `Landing Page.html` - Página principal.
- `Landing Page (standalone source).html` - Versión independiente.
- `Punto Fast Landing.html` - Otra variante del sitio.
- `images/` - Imágenes usadas en la landing.
- `netlify-build/` - Build listo para despliegue en Netlify.
- `uploads/` - Recursos adicionales y capturas.

## Uso

1. Abrir cualquiera de los archivos HTML en un navegador.
2. Subir la carpeta a un host estático o a Netlify.
